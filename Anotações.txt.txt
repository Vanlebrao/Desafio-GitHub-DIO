O Git foi criado em 2005, para armazenar o código.
O Software é feito de forma colaborativa, não realizada apenas por uma pessoa, então entendeu a necessidade de criar um sistema que criasse versões.
Git e Github são diferentes.
Benefícios de aprender sobre o Git:

 1 Controle de versão;
 2 Armazenamento na nuvem;
 3 Trabalho em equipe;
 4 Melhorar seu código;
 5 Reconhecimento;
